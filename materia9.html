Programación II

En la materia de programación II, se adquirio el conocimiento de paradigmas
como: paradigma secuencial, paradigma estructural, paradigma modular,
paradigma orientada a objetos.

Hola mundo

El objetivo principal de un "Hola mundo" es verificar que el entorno de desarrollo
esté configurado correctamente, y que el lenguaje de programación
esté instalado adecuadamente.


#include <iostream>

int main() {
    std::cout << "Hola mundo";
    return 0;
}

Variable - Constante

Una variable es un espacio de memoria con un nombre específico que se utiliza
para almacenar datos, que pueden cambiar durante la ejecución del programa.
Una constante es similar a una variable con la diferencia que su valor
no cambia durante la ejecución de un programa.


#include <iostream>

int main() {
    // Variable
    int edad;
    edad = 30;
    std::cout << "\nLa edad es: " << edad;

    edad = 31;
    std::cout << "\nAhora la edad es: " << edad;

    // Constante
    const int gestion = 2024; 
    std::cout << "\nLa gestion es: " << gestion;

    return 0;
}

Tipos de datos

Los tipos de datos representan los diferentes tipos de valores que pueden ser
almacenados y manipulados en un programa.


#include <iostream>

int main() {
    // Tipo de dato entero         
    short usuario = 2609;
    int ci = 8505484;
    long long numero = 64846140LL; 

    // Tipo de dato flotante o decimal
    float peso = 70.4f;
    double altura = 1.73;

    // Tipo de dato caracter
    char inicial = 'L';

    // Tipo de dato booleano
    bool mayorDeEdad = true;
    
    // Imprimir los resultados
    std::cout << "\nUsuario: " << usuario <<
                 "\nCI: " << ci <<
                 "\nNumero: " << numero <<
                 "\nPeso: " << peso <<
                 "\nAltura: " << altura <<
                 "\nInicial: " << inicial <<
                 "\nMayor de edad: " << (mayorDeEdad ? "true" : "false") << std::endl;

    return 0;
}

Operadores aritmeticos

Son símbolos especiales que se utilizan para realizar operaciones matemáticas
sobre variables y valores.


#include <iostream>

int main() {
    int numero1;
    int numero2;
    int suma, resta, producto, division;

    std::cout << "Ingrese primer numero: ";
    std::cin >> numero1;
    std::cout << "Ingrese segundo numero: ";
    std::cin >> numero2;

    // Operador de adición
    suma = numero1 + numero2;

    // Operador de sustracción
    resta = numero1 - numero2;

    // Operador de multiplicación
    producto = numero1 * numero2;

    // Operador de división
    division = numero1 / numero2;

    std::cout << "\nLa suma de los dos numeros ingresados es: " << suma <<
                 "\nLa resta de los dos numeros ingresados es: " << resta <<
                 "\nEl producto de los dos numeros ingresados es: " << producto <<
                 "\nLa division de los dos numeros ingresados es: " << division << std::endl;

    return 0;
}

Operadores de asignación

Son asignaciones que se dan a valores a variables.
Además de la asignación básica (=),
existen operadores compuestos que combinan una operación aritmética con la asignación.


#include <iostream>

int main() {
    int numero = 10; 
    std::cout << "El valor de numero es: " << numero << std::endl;

    // Operador +=
    numero += 1; 
    std::cout << "Ahora el valor de numero es: " << numero << std::endl;

    // Operador -=
    numero -= 2; 
    std::cout << "Ahora el valor de numero es: " << numero << std::endl;

    // Operador *=
    numero *= 3; 
    std::cout << "Ahora el valor de numero es: " << numero << std::endl;

    // Operador /=
    numero /= 4; 
    std::cout << "Ahora el valor de numero es: " << numero << std::endl;

    return 0;
}

Operadores relacionales

Son símbolos utilizados para comparar dos valores y determinar la relación entre ellos.
Devuelven un resultado booleano (true o false) según se cumpla la condición
especificada.


#include <iostream>

int main() {
    int edadLuis = 25;
    int edadSilvia = 30;

    // Activar la representación booleana como true/false
    std::cout << std::boolalpha;

    // Operador ==
    std::cout << "¿La edad de Luis es igual a la edad de Silvia?" << std::endl;
    std::cout << (edadLuis == edadSilvia) << std::endl;

    // Operador !=
    std::cout << "¿La edad de Luis es diferente a la edad de Silvia?" << std::endl;
    std::cout << (edadLuis != edadSilvia) << std::endl;

    // Operador >
    std::cout << "¿La edad de Luis es mayor que la edad de Silvia?" << std::endl;
    std::cout << (edadLuis > edadSilvia) << std::endl;

    // Operador <
    std::cout << "¿La edad de Luis es menor que la edad de Silvia?" << std::endl;
    std::cout << (edadLuis < edadSilvia) << std::endl;

    // Operador >=
    std::cout << "¿La edad de Luis es mayor o igual que la edad de Silvia?" << std::endl;
    std::cout << (edadLuis >= edadSilvia) << std::endl;

    // Operador <=
    std::cout << "¿La edad de Luis es menor o igual que la edad de Silvia?" << std::endl;
    std::cout << (edadLuis <= edadSilvia) << std::endl;

    return 0;
}

Operadores lógicos

Son herramientas fundamentales que permiten combinar o modificar
valores booleanos para realizar evaluaciones más complejas.


#include <iostream>

int main() {
    bool esEstudiante = true;
    int edad = 18;

    // Operador AND (&&)
    bool esMayorDeEdadYEstudiante = (edad >= 18) && esEstudiante;
    std::cout << "¿Es mayor de edad y estudiante? \n" << std::boolalpha << esMayorDeEdadYEstudiante << std::endl;

    // Operador OR (||)
    bool esMayorDeEdadOEstudiante = (edad >= 18) || esEstudiante;
    std::cout << "¿Es mayor de edad o estudiante? \n" << std::boolalpha << esMayorDeEdadOEstudiante << std::endl;

    // Operador NOT (!)
    bool noEsEstudiante = !esEstudiante;
    std::cout << "¿No es estudiante? \n" << std::boolalpha << noEsEstudiante << std::endl;

    return 0;
}

Estructura de control
Condicional If - Else

Es una condicional que permite ejecutar diferentes bloques de código
según el resultado de una condición booleana.
Esto significa que además de ejecutar un bloque de código cuando la condición es verdadera (if),
también se puede especificar qué hacer cuando la condición es falsa (else).


#include <iostream>

int main() {
    int edad;

    std::cout << "Ingrese su edad: ";
    std::cin >> edad;

    // Condicional If - Else
    if (edad >= 18) {
        std::cout << "Eres mayor de edad." << std::endl;
    } else {
        std::cout << "Eres menor de edad." << std::endl;
    }

    return 0;
}

Estructura de control
Condicional Switch - Case

Es una condicional para evaluar el valor de una variable o expresión ,
y ejecutar diferentes bloques de código según el valor de esa variable.
Es una alternativa a la serie de instrucciones if-else if-else
cuando se tienen múltiples condiciones a evaluar sobre una misma variable.


#include <iostream>

int main() {
    int dia;
    std::string nombreDia;

    std::cout << "Ingrese un numero del 1 al 7: ";
    std::cin >> dia;

    // Condicional Switch - Case
    switch (dia) {
        case 1:
            nombreDia = "Lunes";
            break;
        case 2:
            nombreDia = "Martes";
            break;
        case 3:
            nombreDia = "Miercoles";
            break;
        case 4:
            nombreDia = "Jueves";
            break;
        case 5:
            nombreDia = "Viernes";
            break;
        case 6:
            nombreDia = "Sabado";
            break;
        case 7:
            nombreDia = "Domingo";
            break;
        default:
            nombreDia = "Dia no válido";
            break;
    }

    std::cout << "El dia " << dia << " es " << nombreDia << std::endl;

    return 0;
}

Estructura de control
Condicional For

Es una construcción de bucle que permite ejecutar un bloque de código
un número específico de veces.


#include <iostream>

int main() {
    int suma = 0;

    // Condicional For
    for (int i = 1; i <= 10; i++) {
        suma += i;
        std::cout << suma << std::endl;
    }

    std::cout << "La suma de los primeros 10 numeros es: " << suma << std::endl;

    return 0;
}

Estructura de control
Condicional While

Es una construcción de bucle que se repite mientras una condición específica sea verdadera.


#include <iostream>

int main() {
    int numero = 1;

    while (numero <= 5) {
        std::cout << "Numero: " << numero << std::endl;
        numero++;
    }

    std::cout << "El valor de numero es: " << numero << std::endl;

    return 0;
}

Estructura de control
Condicional Do - While

Es una construcción de bucle similar a la estructura while,
pero con una diferencia fundamental la condición se evalúa al final del bucle
en lugar de al principio.


#include <iostream>
using namespace std;

int main() {
    int numero;

    do {
        cout << "Ingrese un numero mayor a cero: ";
        cin >> numero;
    } while (numero <= 0);

    cout << "Numero valido ingresado: " << numero << endl;

    return 0;
}
