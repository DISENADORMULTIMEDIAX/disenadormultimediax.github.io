Variable - Constante

Una variable es un espacio de memoria con un nombre específico que se utiliza para almacenar datos,
que pueden cambiar durante la ejecución del programa.
Un constante es un valor que no cambia durante la ejecución de un programa.


public class Main {
    public static void main(String[] args) {

        // Variable
        int edad;
        edad = 30;
        System.out.println("La edad es: " + edad);

        edad = 31;
        System.out.println("Ahora la edad es: " + edad);

        // Constante
        final int  gestion = 2024;;
        System.out.println("La gestion es: " + gestion);

        gestion = 2025;
        System.out.println("Ahora la gestion es: " + gestion);
        // Al ser una constante no se puede modificar su valor por ende el IDE nos mostrara error.
    }
}

Operadores aritmeticos

Son símbolos especiales que se utilizan para realizar operaciones matemáticas sobre variables y valores.


import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner e=new Scanner(System.in);
        int numero1;
        int numero2;
        int suma,resta,producto,division;

        System.out.print("Ingrese primer numero: ");
        numero1=e.nextInt();
        System.out.print("Ingrese segundo numero: ");
        numero2=e.nextInt();

        // Operador de adición
        suma=numero1+numero2;

        // Operador de sustracción
        resta=numero1-numero2;

        // Operador de multiplicación
        producto=numero1*numero2;

        // Operador de división
        division=numero1/numero2;

        System.out.println("\nLa suma de los dos numeros ingresados es: "+ suma +
                           "\nLa resta de los dos numeros ingresados es: "+ resta +
                           "\nEl producto de los dos nu1meros ingresados es: " + producto +
                           "\nLa division de los dos numeros ingresados es: "+ division );
    }
}

Operadores de asignación

Son asignaciones que se dan a valores a variables.
Además de la asignación básica (=),
existen operadores compuestos que combinan una operación aritmética con la asignación.


public class Main {
    public static void main(String[] args) {
        int numero = 10;
        System.out.println("El valor de numero es: " + numero);

        // Operador +=
        numero = numero + 1;
        System.out.println("Ahora el valor de numero es: " + numero);

        // Operador -=
        numero = numero - 2;
        System.out.println("Ahora el valor de numero es: " + numero);

        // Operador *=
        numero = numero * 3;
        System.out.println("Ahora el valor de numero es: " + numero);

        // Operador /=
        numero = numero / 4;
        System.out.println("Ahora el valor de numero es: " + numero);
    }
}

Operadores relacionales

Son símbolos utilizados para comparar dos valores y determinar la relación entre ellos.
Devuelven un resultado booleano (true o false) según se cumpla la condición especificada.


public class Main {
    public static void main(String[] args) {
        int edadLuis = 25;
        int edadSilvia = 30;

        // Operador ==
        System.out.println("¿La edad de Juan es igual a la edad de María?");
        System.out.println(edadLuis == edadSilvia);

        // Operador !=
        System.out.println("¿La edad de Juan es diferente a la edad de María?");
        System.out.println(edadLuis != edadSilvia);

        // Operador >
        System.out.println("¿La edad de Juan es mayor que la edad de María?");
        System.out.println(edadLuis > edadSilvia);

        // Operador <
        System.out.println("¿La edad de Juan es menor que la edad de María?");
        System.out.println(edadLuis < edadSilvia);

        // Operador >=
        System.out.println("¿La edad de Juan es mayor o igual que la edad de María?");
        System.out.println(edadLuis >= edadSilvia);

        // Operador <=
        System.out.println("¿La edad de Juan es menor o igual que la edad de María?");
        System.out.println(edadLuis <= edadSilvia);
    }
}

Operadores lógicos

Son herramientas fundamentales que permiten combinar o modificar valores booleanos para realizar evaluaciones más complejas.


public class Main {
    public static void main(String[] args) {
        boolean esEstudiante = true;
        int edad = 18;

        //Operador AND (&&)
        boolean esMayorDeEdadYEstudiante = edad >= 18 && esEstudiante;
        System.out.println("¿Es mayor de edad y estudiante? \n" + esMayorDeEdadYEstudiante);

        //Operador OR ()
        boolean esMayorDeEdadOEstudiante = edad >= 18  esEstudiante;
        System.out.println("¿Es mayor de edad o estudiante? \n" + esMayorDeEdadOEstudiante);

        //Operador NOT (!)
        boolean noEsEstudiante = ! esEstudiante;
        System.out.println("¿No es estudiante? \n" + noEsEstudiante);
    }
}

Estructura de control
Condicional If - Else

Es una condicional que permite ejecutar diferentes bloques de código según el resultado de una condición booleana. Esto significa que además de ejecutar un bloque de código cuando la condición es verdadera (if), también se puede especificar qué hacer cuando la condición es falsa (else).


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner e = new Scanner(System.in);

        System.out.print("Ingrese su edad: ");
        int edad = e.nextInt();

        // Condicional If - Else
        if (edad >= 18) {
            System.out.println("Eres mayor de edad.");
        } else {
            System.out.println("Eres menor de edad.");
        }
    }
}

Estructura de control
Condicional Switch - Case

Es una condicional para evaluar el valor de una variable o expresión ,
y ejecutar diferentes bloques de código según el valor de esa variable.
Es una alternativa a la serie de instrucciones if-else if-else cuando se tienen múltiples condiciones a evaluar sobre una misma variable.


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner e = new Scanner(System.in);

        System.out.print("Ingrese un número del 1 al 7: ");
        int dia = e.nextInt();
        String nombreDia;

        // Condicional Switch - Case
        switch (dia) {
            case 1:
                nombreDia = "Lunes";
                break;
            case 2:
                nombreDia = "Martes";
                break;
            case 3:
                nombreDia = "Miércoles";
                break;
            case 4:
                nombreDia = "Jueves";
                break;
            case 5:
                nombreDia = "Viernes";
                break;
            case 6:
                nombreDia = "Sábado";
                break;
            case 7:
                nombreDia = "Domingo";
                break;
            default:
                nombreDia = "Día no válido";
                break;
        }

        System.out.println("El día " + dia + " es " + nombreDia);
    }
}

Estructura de control
Condicional For

Es una construcción de bucle que permite ejecutar un bloque de código un número específico de veces.


public class Main {
    public static void main(String[] args) {
        int suma = 0;

        // Condicional For
        for (int i = 1; i <= 10; i++) {
            suma += i;
            System.out.println(suma);
        }
        System.out.println("La suma de los primeros 10 números es: " + suma);

    }
}

Estructura de control
Condicional While

Es una construcción de bucle que se repite mientras una condición específica sea verdadera.


public class Main {
    public static void main(String[] args) {
        int numero = 1;

        while (numero <= 5) {
            System.out.println("Número: " + numero);
            numero++;
        }

        System.out.println("El valor de numero es: " + numero);
    }
}

Estructura de control
Condicional Do - While

Es una construcción de bucle similar a la estructura while,
pero con una diferencia fundamental: la condición se evalúa al final del bucle en lugar de al principio.


import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner e = new Scanner(System.in);
        int numero;

        do {
            System.out.print("Ingrese un número mayor a cero: ");
            numero = e.nextInt();
        } while (numero <= 0);

        System.out.println("Número válido ingresado: " + numero);
    }
}
