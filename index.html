Operadores aritmeticos

Para este proyecto se uso 2 variables para la entrada de los valores,
para luego aplicar operadores aritmeticos, y posteriormente mostrar el resultado.
El programa es una calculadora que realiza operaciones de adición, sustracción, producto y división.


import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner e=new Scanner(System.in);
        int numero1;
        int numero2;
        int suma,resta,producto,division;
        
        System.out.print("Ingrese primer numero: ");
        numero1=e.nextInt();
        System.out.print("Ingrese segundo numero: ");
        numero2=e.nextInt();
        
        suma=numero1+numero2;
        resta=numero1-numero2;
        producto=numero1*numero2;
        division=numero1/numero2;
        
        System.out.println("La suma de los dos numeros ingresados es: "+suma);
        System.out.println("La resta de los dos numeros ingresados es: "+resta);
        System.out.println("El producto de los dos numeros ingresados es: "+producto);
        System.out.println("La division de los dos numeros ingresados es: "+division);
    }
}
